## `const createArrArgMatch = ctx => function arrArgMatch(...tests)`

<h3>
  Array match
</h3> Return a function that checks to see if an array contains either any of the values listed or if any of the predicate functions provided return true when run over all items in the array. Eg,
<pre>

arrArgMatch(isNumeric, isNumeric, isNumeric)([1,2,3]); // true arrArgMatch(isNumeric, isNumeric, isNumeric, '...')([1,2,3]); // true arrArgMatch(isString, isNumeric, isNumeric, '...')([1,2,3]); // false arrArgMatch(isString, isNumeric, isNumeric, '...')(['1',2,3]); // true arrArgMatch(isNumeric, isNumeric, isNumeric, '...')([1,2,3,4]); // true arrArgMatch(1, 2)([1,3]); // false </code></pre>

- **Parameters:** `tests` — `...function|*` — Either values, `['...', predicate]` or predicate functions used to test the contents of the array.
- **Returns:** `function` — A function of the form <code>{array => boolean}</code>

## `const createArrTypeMatch = ctx => function arrTypeMatch(test)`

<h3>
  Array type match
</h3> Return a function that checks to see if an array contains only the values listed or if the predicate function provided returns true when run over all items in the array. Eg,
<pre>

arrTypeMatch(isNumeric)([1,2,3]); // true arrTypeMatch(isNumeric)([1,2,'3']); // false arrTypeMatch(isNumeric)([]); // true </code></pre>

- **Parameters:** `test` — `function|*` — predicate function used to test the contents of the array.
- **Returns:** `function` — A function of the form <code>{array => boolean}</code>

## `const createPrim = ctx => function prim(primative)`

<h3>
  Primative predicate
</h3> Forms a predicate from a given JavaSCript primative object to act as a typeof check for the input value.

Eg. <pre><code> prim(Function)(() => {}); // true prim(Number)(6); // true </code></pre>

- **Parameters:** `primative` — `object` — The input primative one of Array, Boolean, Number, Symbol, BigInt, String, Function, Object
- **Returns:** `function` — A function of the form <code>{any => boolean}</code>
